# ALONE PROJECT FRONT

## ALONE 이란? (어떤 서비스인가)

- 새로운 장소에서 새로운 사람들을 쉽게 만나게 도와주는 서비스
- 함께 여행할 사람들을 찾는 서비스

## FRONT 구현 기술

- web framework: express, webpackDevServer(dev)
- js library: React(component), MobX(status)
- Bundler: webpack
- css, component: sematic-ui
